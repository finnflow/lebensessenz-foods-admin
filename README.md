# Lebensessenz · Lebensmittel-Pflege (Browser-Oberfläche)

Statische Pflege-Oberfläche für die Lebensmittel-Datenbank. Redet direkt mit Supabase;
Zugriff nur nach **Login** (Daten durch Supabase-Auth + Row Level Security geschützt).
Enthält **keine Geheimnisse** — nur die öffentliche Supabase-URL und den Publishable-Key.

Gehostet über GitHub Pages. Anmeldung mit dem in Supabase angelegten Nutzer.
